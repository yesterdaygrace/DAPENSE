# Architecture Summary

## Project
wasas — Laravel-based financial/accounting system.

## Stack
- PHP (Laravel)
- Blade templates
- Tailwind CSS
- MySQL

## Key Modules
- Jurnaling (Journaling)
- BukuBesar (General Ledger)
- NeracaSaldo (Trial Balance)
- Laporan (Reports): Neraca, ArusKas, AsetNeto, Investasi, etc.
- Multi-role exports (admin, bod, operator, rootsuperuser)
